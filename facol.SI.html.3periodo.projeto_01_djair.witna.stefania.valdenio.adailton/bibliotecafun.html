Array

    Retorna um array de parâmetros. Índices podem ser atribuídos aos parâmetros através do operador.
    Array array(...);
    Exemplo:
    $arr = array(“palavras” => array(“1” => “index”,”2” => “main“,”3” =>
    “default”),”tradução” => array(“índice”,”principal”,”padrão”) );
